# scrapy-nivea
Scraper that collects products from Nivea main product page.

Spider uses pagnation in odred to get all 475 items.

Data needed to be extracted:

- name of product
- category
- link to the item
- rating
- number of reviews

https://www.nivea.hr/proizvodi
